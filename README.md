### Installation :
Open Terminal, and clone this repo:  
> git clone https://github.com/harryandriyan/warta-scrap

Go to project folder
> cd warta-scrap

Setup virtualenv
> virtualenv venv

Activate virtualenv
> . venv/bin/activate

Install requirements
> pip install -r requirements.txt
